# Quick Music Download

<img align="left" height="65vw" src="https://i.imgur.com/Xk737o8.png">

Part of the Music Download Suite. It takes a YouTube or SoundCloud URL as input and downloads that music file.

Download Here: [https://www.icloud.com/shortcuts/6366692993ae47c4a5b736a4502b6864](https://www.icloud.com/shortcuts/6366692993ae47c4a5b736a4502b6864)

---

### Description

This Shortcut takes URLs as input. If no url is passed to the input, it will first look at the clipboard, then the current content on the screen, and finally it will ask the user for the URL if it failed to find one from those places first. The URL needs to be a valid YouTube or SoundCloud link that points to a video/audio.

This Shortcut will ask to be updated when it needs to. Simply click Update when it asks, or force it to update by passing ```--update``` as the input instead of a URL.

> [!NOTE]
> This Shortcut can only be run on a Mac.
